## ü•ê Alternative: Croissant Wallet (2 minutes)

### Step 1: Install Croissant
1. Go to: https://croissant.linera.io
2. Click "Install"
3. Add to browser

### Step 2: Create Wallet
1. Open Croissant extension
2. Create new wallet
3. Save your seed phrase!

### Step 3: Connect in Game
1. Open chess game
2. Click "Connect Wallet"
3. Select "Croissant"
4. Click "Connect Croissant"
5. Approve in popup

**Done!** Start playing.

A: No! Testnet Conway gives free test tokens automatically.

## üÜò Quick Troubleshooting

**Can't connect?**

- Web Client auto-connects correctly
